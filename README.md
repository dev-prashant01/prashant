# Prashant — MERN Stack Developer Portfolio

## Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: React Icons
- **Fonts**: Space Mono + Syne (Google Fonts)

## Features

- 🎯 Custom animated cursor with lag ring
- ⌨️ Typewriter hero cycling through roles
- 🎬 Scroll-triggered reveal animations (Framer Motion)
- 📱 Fully responsive (mobile → desktop)
- 🎨 Dark terminal aesthetic with orange + teal accents
- ⚡ Skill bars with animated fill
- 🃏 Project cards with gradient thumbnails
- 📋 One-click email copy in Contact section
- 🔢 MERN stack highlight section
